Antônia gosta de fazer artesanato. Hoje ela contornou o molde de uma peça que vai produzir. Com uma tesoura, recortou o tecido.

Vamos explorar essa situação...

1. Sabendo que as medidas indicadas no tecido estão em centímetros, quantos centímetros a tesoura percorreu para recortar a peça?

Com base na situação, podemos concluir que:

Ao medir o contorno de uma quadra, de um quarto, de uma horta, de uma casa, de um tecido, de uma figura, determinamos o perímetro.

Perímetro é uma medida de comprimento relacionada ao contorno de uma figura.

O perímetro de qualquer polígono é a soma das medidas de seus lados.

[(2 x 2,50 + 2 x 3,30) – 2 x 0,90] =

[11,60 – 2 x 0,90] =

Ao calcularmos o contorno ou o perímetro de uma figura, devemos verificar se todas as medidas de comprimento estão expressas na mesma unidade. Caso não estejam, antes de somá-las precisamos primeiro fazer as transformações de medida. Todas devem estar representadas com a mesma unidade. Só após esta verificação e transformação (caso necessário) é que podemos somar.
